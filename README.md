# Currency_Converter
A simple and responsive Currency Converter Web Application that allows users to convert currencies using real-time exchange rates from an API.

## ✨ Features

### 🌎 Real-Time Currency Conversion
- Fetches the latest exchange rates using the ExchangeRate API.

### 💵 Multiple Currency Support
- Convert between different currencies using dynamic dropdown menus.

### 🔄 Currency Swap Function
- Quickly switch the "From" and "To" currencies with one click.

### 🚩 Dynamic Country Flags
- Displays the correct country flag based on the selected currency.

### ⚡ API Integration
- Uses JavaScript Fetch API to retrieve and process exchange rate data.

### ⚠️ Error Handling
- Displays an error message if the API request fails or the user is offline.

## 🛠️ Technologies Used

- 🧱 HTML
- 🎨 CSS
- ⚡ JavaScript
- 🌐 ExchangeRate API

## 🚀 How to Run

### 1. Clone this repository

### 2. Open index.html in your browser

### 3. Enter an amount and select currencies to convert
